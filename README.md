# musicapp

一个基于vue3的网易云音乐移动端的前端练习项目

后端接口由[网易云音乐 NodeJS 版 API](https://github.com/Binaryify/NeteaseCloudMusicApi)提供

目前所有组件均为js手写，下一个正式项目会用上element-plus
